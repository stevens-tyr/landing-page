Save hours of your time grading

Tyr is a scalable assignment grader that helps you to grade assignments accurately and effortlessly. We can offload your time grading so you can focus on helping students.

Built using Kubernetes

Flexible

Set up Tyr grading servers either on the cloud or host them yourself. Tyr allows you to host it however you want.

Fault Tolerant

Worried about faulty code bringing down our test server? It won't happen because Tyr uses Kubernetes which, by design, will provide a robust architect that is resistant to total failure.

Instant Feedback

Facilitate learning by enabling instant feedback on homeworks. Students will better learn if they are able to ask questions about an assignment when it's still fresh in their heads.
